# FugoTools

FugoTools是一个土木工程设计辅助工具集合，主要用于石化行业常见构筑物的简单前期计算估料，基于Python和PySide6开发。

## 功能特点

### 插件列表
- **块式基础计算** - 提供块式基础的设计计算
- **管墩计算** - 管墩设计计算，包括地基承载力验算、抗倾覆计算等，支持导出计算书和料表
- **型钢特性表** - 型钢截面特性查询和计算
- **YJK柱脚内力处理工具** - YJK软件柱脚内力数据处理

### 技术特点
- 基于PySide6的现代化GUI界面
- 插件化架构，易于扩展
- 详细的计算过程和结果展示

### 环境要求
- Python 3.8+
- PySide6
- 其他依赖包（见requirements.txt）

## 使用方法
1. 启动应用后，在主界面选择需要的工具插件
2. 输入相关参数，点击计算按钮
3. 查看详细的计算过程和结果
4. 根据需要调整参数重新计算

## 项目结构
- `core/` - 核心功能模块
- `plugins/` - 插件目录
- `ui/` - 用户界面模块
- `resources/` - 资源文件

## 开发说明
如需添加新插件，请参考`plugins/base_plugin.py`中的基类实现，遵循现有的插件架构。
